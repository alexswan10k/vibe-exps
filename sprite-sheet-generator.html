Sprite Sheet Generator

Instructions:

  1. Paste images from clipboard (Ctrl+V or Cmd+V)
  2. Set grid size (columns and rows)
  3. Set tile size (width and height in pixels)
  4. Name each image in the list below
  5. Use ↑↓ arrows to reorder, X to delete images
  6. Sprite sheet updates automatically in real-time
  7. Click "Save as PNG" to download the sprite sheet
  8. View simplified JSON output with name and position
  9. Click "Show TypeScript Types" for type definitions
